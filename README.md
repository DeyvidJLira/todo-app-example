## Regras de Negócio

- O título de uma tarefa deve ter no mímimo 3 e no máximo 40 caractéres;
- A prioridade de uma tarefa vai de 0 (mais baixa) à 9 (mais alta);
- Um tarefa pode ser marcada como finalizada.
